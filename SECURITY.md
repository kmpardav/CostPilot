# Security Policy

## Reporting a Vulnerability
If you believe you have found a security issue in CostPilot, please email
Bardavouras Konstantinos at kostasbardavouras@gmail.com with details. Include:
- A description of the issue and the potential impact
- Steps to reproduce or proof-of-concept code
- Any relevant logs or configuration details

We aim to acknowledge reports within 3 business days. If the issue is
validated, we will work with you on a fix timeline and coordinate disclosure.

Please do not publicly disclose vulnerabilities until we have confirmed and
released a remediation or provided guidance.
